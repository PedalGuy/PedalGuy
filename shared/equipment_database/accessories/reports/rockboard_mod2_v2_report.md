## 器材概述

### 基本資訊

**品牌:** RockBoard
**製造商:** Warwick
**型號:** MOD 2 V2
**全名:** RockBoard MOD 2 V2 All-in-One Pedalboard Patchbay
**類型:** Pedalboard 配件 (Patchbay Module)
**製造國:** Germany (推測)
**發行年份:** 2020
**建議售價:** $179 USD (預估)
**街價:** $150-180 USD

### 核心賣點

RockBoard MOD 2 V2 是一款專為 Pedalboard 設計的全功能 Patchbay 模組，可以統一管理訊號線、MIDI、USB 和電源連接。這款模組採用 **前後鏡像連接設計**，前面板連接效果器，背面連接音箱/電源，讓 Pedalboard 的接線更加整潔有序。

**獨特之處:**
- **4 x TRS thru** - 比 MOD 3 V2 多 1 個 TRS 接口
- **2 x MIDI thru** - MOD 3 V2 沒有 MIDI (⭐ 核心優勢)
- **1 x USB Type A thru** - MOD 3 V2 沒有 USB
- **所有連接器前後鏡像** - 前面板 = 後面板，簡化接線
- **模組化設計** - 可安裝在 RockBoard Pedalboard 的 MOD slot
- **可拆卸前面板** - 彈性安裝方式

### 適合的使用場景

| 場景 | 適配度 | 說明 |
|------|--------|------|
| **MIDI 控制效果器** | ⭐⭐⭐⭐⭐ | 2 個 MIDI thru，控制 PA-1QG, ODL-1-CS 等 |
| **前級效果器 Pedalboard** | ⭐⭐⭐⭐⭐ | 管理 Gain Pedals, Compressor, EQ 接線 |
| **雙訊號鏈配置** | ⭐⭐⭐⭐⭐ | 配合 Empress Buffer++ 切換 Rock/Clean 訊號鏈 |
| **四線法 (4CM) 預留** | ⭐⭐⭐⭐ | 4 個 TRS 足夠處理四線法接口 |
| **USB MIDI Controller** | ⭐⭐⭐⭐ | USB Type A thru 支援 USB 設備 |
| **XLR 錄音需求** | ⭐ | 無 XLR 接口，不適合 (選 MOD 3 V2) |

**目標用戶:**
- 需要 MIDI 控制效果器的玩家 (PA-1QG 99 preset 切換)
- 主 Pedalboard 放前級效果器，空間系分離配置
- 使用 Empress Buffer++ 雙訊號鏈配置
- 不需要 XLR 連接的玩家

### 連接器配置

#### **TRS 1/4 Inch 接口 (⭐ 核心功能)**

| 項目 | 規格 |
|------|------|
| **數量** | 4 x TRS thru |
| **類型** | 6.3 mm / 1/4 inch TRS |
| **配置** | 前面板接口 = 後面板接口 (鏡像) |

**用途:**
- 吉他/效果器訊號線路管理
- 效果器鏈跳線整理
- Pedalboard 輸入/輸出整合
- 四線法 (4CM) 預留接口

**連接範例:**
- **TRS 1:** 吉他 Input
- **TRS 2:** 主 Pedalboard Output → 音箱/空間系
- **TRS 3:** 預留四線法 (音箱 FX Send → 空間系)
- **TRS 4:** 預留四線法 (空間系 Output → 音箱 FX Return)

---

#### **MIDI 接口 (⭐ 核心優勢)**

| 項目 | 規格 |
|------|------|
| **數量** | 2 x MIDI thru |
| **類型** | 5-pin DIN MIDI |
| **配置** | 前面板接口 = 後面板接口 (鏡像) |

**用途:**
- MIDI Controller → MIDI 效果器
- MIDI 訊號鏈管理
- 控制支援 MIDI 的效果器 (例: PA-1QG, ODL-1-CS)

**連接範例:**
- **MIDI 1:** MIDI Controller → PA-1QG MIDI In
- **MIDI 2:** PA-1QG MIDI Thru → ODL-1-CS MIDI In

**⚠️ 重要提示:** 這是 MOD 2 V2 相對於 MOD 3 V2 的核心優勢！MOD 3 V2 沒有 MIDI 接口。

## 功能特性

### 核心功能 1: 前後鏡像連接

**描述:** 所有連接器前後鏡像，前面板連接效果器，背面連接音箱/電源

**好處:**
- 簡化接線邏輯
- 訊號線、MIDI、電源統一整理
- Pedalboard 外觀更整潔

**範例:**
```
[前面板] 吉他 → MOD 2 TRS 1 → [後面板] → Pedalboard 內部效果器
[前面板] Pedalboard 輸出 ← MOD 2 TRS 2 ← [後面板] ← 音箱 Input
```

---

### 核心功能 2: MIDI 控制系統 (⭐ 核心優勢)

**描述:** 2 個 MIDI thru 可以串接多個 MIDI 設備

**應用場景:**
- MIDI Controller 控制 PA-1QG 99 preset
- PA-1QG MIDI Thru → ODL-1-CS MIDI In
- 集中管理 MIDI 訊號鏈

**連接範例:**
```
MIDI Controller (背面) → MOD 2 MIDI 1 (後面板)
   ↓
MOD 2 MIDI 1 (前面板) → PA-1QG MIDI In
   ↓
PA-1QG MIDI Thru → MOD 2 MIDI 2 (前面板)
   ↓
MOD 2 MIDI 2 (前面板) → ODL-1-CS MIDI In
```

---

### 核心功能 3: 訊號線整理

**描述:** 4 個 TRS thru 可以處理多種訊號路徑

**應用場景:**
- **TRS 1:** 吉他 Input
- **TRS 2:** 主 Pedalboard Output → 音箱/空間系
- **TRS 3:** 預留四線法 (音箱 FX Send → 空間系)
- **TRS 4:** 預留四線法 (空間系 Output → 音箱 FX Return)

**好處:**
- 4 個 TRS 足夠處理雙訊號鏈 + 四線法預留
- 比 MOD 3 V2 多 1 個 TRS，更有彈性

---

### 核心功能 4: 模組化設計

**描述:** 可安裝在 RockBoard Pedalboard 的 MOD slot

**好處:**
- 與 Pedalboard 完美整合
- 可拆卸前面板，彈性安裝
- 適用於 RockBoard 或其他品牌 Pedalboard

**安裝方式:**
- 安裝在 Pedalboard 頂部 (含前面板)
- 安裝在 Pedalboard 底部 (拆除前面板)

## 使用範例

### 範例 1：前級效果器 Pedalboard + MIDI 控制

**音樂風格:** 任何風格
**適合吉他:** 任何吉他

**配置:**

```
🎸 吉他 (背面)
   ↓
【MOD 2 TRS 1 Input (後面板)】
   ↓
MOD 2 TRS 1 (前面板) → Pedalboard 效果器鏈
   ↓
Pedalboard Output → MOD 2 TRS 2 (前面板)
   ↓
【MOD 2 TRS 2 Output (後面板)】→ 音箱 Input (背面)

【MIDI 控制】
MIDI Controller (背面) → MOD 2 MIDI 1 (後面板)
   ↓
MOD 2 MIDI 1 (前面板) → PA-1QG MIDI In
   ↓
PA-1QG MIDI Thru → MOD 2 MIDI 2 (前面板)
   ↓
MOD 2 MIDI 2 (前面板) → ODL-1-CS MIDI In
```

**效果:**
- 4 個 TRS 處理訊號線
- 2 個 MIDI 控制效果器 Preset
- 統一線路管理

**好處:**
- 可用 MIDI Controller 切換 PA-1QG 99 preset
- 集中管理所有接線
- Pedalboard 外觀整潔

---

### 範例 2：Rock + Clean 雙訊號鏈 (Buffer++ 配置)

**音樂風格:** Rock, Blues, Jazz, Neo Soul
**適合吉他:** 任何吉他

**配置:**

```
🎸 吉他 (背面)
   ↓
【MOD 2 TRS 1 Input (後面板)】
   ↓
MOD 2 TRS 1 (前面板) → ① Empress Buffer++ Input
   ↓
Buffer++ Loop 1 Send → [Rock 訊號鏈] → Loop 1 Return
   - Cali76 FET (Compressor)
   - Roshi Blacklon (Amp-in-a-Box)
   - TWA Source Code (TS Evolution)
   - Free the Tone ODL-1-CS (Dumble)
   ↓
Buffer++ Loop 2 Send → [Clean 訊號鏈] → Loop 2 Return
   - Empress MKII (透明壓縮)
   - Mad Professor Sweet Honey (溫暖 OD)
   - PRS Horsemeat (Klon Boost)
   - JHS Morning Glory (BB Overdrive)
   ↓
Buffer++ Output → MOD 2 TRS 2 (前面板)
   ↓
【MOD 2 TRS 2 Output (後面板)】→ 音箱 Input 或 空間系 Pedalboard (背面)

【四線法預留接口】
MOD 2 TRS 3 (後面板): 音箱 FX Send (背面) → MOD 2 TRS 3 (前面板) → 空間系 Pedalboard Input
MOD 2 TRS 4 (前面板): 空間系 Pedalboard Output → MOD 2 TRS 4 (後面板) → 音箱 FX Return (背面)

【MIDI 控制】
MIDI Controller (背面) → MOD 2 MIDI 1 (後面板) → PA-1QG MIDI In
PA-1QG MIDI Thru → MOD 2 MIDI 2 (前面板) → ODL-1-CS MIDI In
```

**Buffer++ 路由模式:** Mode 5 - Loop Select (在 Loop 1 和 Loop 2 之間切換)

**效果器數量:**
- **總計:** 10 顆效果器
- **Loop 1 (Rock):** 4 顆
- **Loop 2 (Clean):** 4 顆
- **Always-on (Shared):** 2 顆 (PA-1QG, 預留)

**連接器使用狀況:**
- **TRS 使用:** 4 個 / 4 個可用 ✅ 完全足夠
  - TRS 1: 吉他 Input
  - TRS 2: 主 Pedalboard Output → 音箱/空間系
  - TRS 3: 預留四線法 (音箱 FX Send → 空間系)
  - TRS 4: 預留四線法 (空間系 Output → 音箱 FX Return)
- **MIDI 使用:** 2 個 / 2 個可用 ✅ 完全足夠
  - MIDI 1: MIDI Controller → PA-1QG
  - MIDI 2: PA-1QG → ODL-1-CS

**好處:**
- ✅ 可用 Buffer++ 在 Rock 和 Clean 訊號鏈之間快速切換
- ✅ PA-1QG 保持在主訊號鏈 (Buffer++ 之前)，提供每把吉他專用 EQ
- ✅ 預留四線法接口連接空間系 Pedalboard
- ✅ MIDI 控制 PA-1QG 99 preset 和 ODL-1-CS
- ✅ 4 個 TRS 足夠處理所有訊號路徑

## 優缺點分析

### 優點 ✅

1. **4 個 TRS thru (比 MOD 3 多 1 個)**
   - 足夠處理雙訊號鏈 + 四線法預留
   - 比 MOD 3 V2 更有彈性

   - **MOD 3 V2 沒有 MIDI**

3. **1 個 USB Type A thru**
   - **MOD 3 V2 沒有 USB**
   - 支援 USB MIDI Controller
   - 預留未來擴充彈性

4. **所有連接器前後鏡像**
   - 簡化接線邏輯

   - Pedalboard 外觀整潔

5. **集中管理訊號線、MIDI、電源**
   - 統一接線入口
   - 避免 Pedalboard 接線混亂

6. **模組化設計**
   - 與 RockBoard Pedalboard 完美整合
   - 可拆卸前面板，彈性安裝
   - 適用於其他品牌 Pedalboard

   - $150-180 USD

8. **輕量化**
   - 420g (0.93 lbs)

### 缺點 ❌

1. **無 XLR 連接器**
   - **不適合需要 XLR 輸出的設置**
   - 如果要整合 Tone King Imperial → Interface，應選 **MOD 3 V2**

2. **需要 RockBoard Pedalboard 或自行安裝**
   - 原生設計給 RockBoard MOD slot

4. **功能過剩 (如果不使用 MIDI)**
   - 如果不需要 MIDI 控制，功能可能過剩

## 總結

### 核心優勢

RockBoard MOD 2 V2 是一款專為前級效果器 Pedalboard 設計的全功能 Patchbay 模組，核心優勢在於 **2 個 MIDI thru** (MOD 3 V2 沒有)、**4 個 TRS thru** (比 MOD 3 多 1 個)，以及 **1 個 USB Type A thru** (MOD 3 V2 沒有)。

這款模組特別適合需要 **MIDI 控制效果器** 的玩家，例如控制 PA-1QG 99 preset 切換、ODL-1-CS 通道切換等。搭配 **Empress Buffer++** 雙訊號鏈配置，可以完美管理 Rock + Clean 雙訊號鏈的接線，4 個 TRS 接口足夠處理主 Pedalboard 輸入/輸出 + 四線法預留接口。

相較於 MOD 3 V2，MOD 2 V2 **沒有 XLR 接口**，因此不適合需要整合 Tone King Imperial → Interface XLR 輸出的設置。但對於主 Pedalboard 只放前級效果器、空間系分離配置的玩家來說，MOD 2 V2 的 MIDI 支援是核心優勢。

---

### 最佳使用建議

1. **MIDI 控制玩家:** 使用 2 個 MIDI thru 控制 PA-1QG 99 preset 和 ODL-1-CS
2. **Buffer++ 雙訊號鏈玩家:** 4 個 TRS 足夠處理 Rock/Clean 訊號鏈 + 四線法預留
3. **前級效果器分離玩家:** 主 Pedalboard 放前級，空間系分離，MIDI 控制前級效果器
4. **USB MIDI Controller 玩家:** 使用 USB Type A thru 連接 USB MIDI 設備

---

### 配置定位

**最佳位置:** Pedalboard 接線整合中樞

**推薦配置 (Rock + Clean 雙訊號鏈):**
```
🎸 吉他 (背面) → MOD 2 TRS 1 (後面板)
   ↓
MOD 2 TRS 1 (前面板) → Empress Buffer++
   ↓
Buffer++ Loop 1 [Rock 訊號鏈: Cali76 FET, Blacklon, Source Code, ODL-1-CS]
Buffer++ Loop 2 [Clean 訊號鏈: Empress MKII, Sweet Honey, Horsemeat, Morning Glory]
   ↓
Buffer++ Output → MOD 2 TRS 2 (前面板)
   ↓
MOD 2 TRS 2 (後面板) → 音箱 Input 或 空間系 Pedalboard (背面)

【四線法預留】
MOD 2 TRS 3/4: 音箱 FX Send/Return ↔ 空間系 Pedalboard

【MIDI 控制】
MIDI Controller → MOD 2 MIDI 1 → PA-1QG → MOD 2 MIDI 2 → ODL-1-CS
```
