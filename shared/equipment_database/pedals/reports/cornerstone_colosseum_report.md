## 器材概述

### 基本資訊

**品牌:** Cornerstone
**型號:** Colosseum Dual Overdrive
**類型:** Overdrive
**風格:** Bluesbreaker + Klon 雙通道組合
**設計方式:** 社群驅動設計（Community-Driven Design）
**製造國:** Italy

### 核心賣點

Cornerstone Colosseum 是一款由社群投票決定設計方向的雙通道 Overdrive 效果器，完美結合 **Marshall Bluesbreaker** 與 **Klon Centaur** 兩大經典電路。這款效果器不僅是簡單的複製品，而是在保留經典音色基礎上，針對兩個電路的傳統弱點進行了重大改良。

**獨特之處:**
- **社群驅動設計** - 由 Cornerstone 社群投票決定功能與設計
- **改良版 Bluesbreaker** - 移除 Clipping Diodes，解決傳統 BB 音量低與音色暗的問題
- **Clean Control** - BB 側可混入乾訊號（源自 Cornerstone Gladio），增加音色透明度
- **Clip Blender** - Klon 側可混合 Germanium 與 Silicon Clipping，提供從經典到現代的連續音色變化
- **Order Toggle** - 可切換兩通道順序（BB→Klon 或 Klon→BB），創造不同音色層次
- **4 個後方接孔** - 可將兩側獨立接入 Loop Switcher，或在兩側之間插入其他效果器

### 適合的音樂風格

| 風格 | 適配度 | 說明 |
|------|--------|------|
| **Blues** | ⭐⭐⭐⭐⭐ | BB 與 Klon 都是 Blues 經典音色 |
| **Rock** | ⭐⭐⭐⭐⭐ | BB 提供 British Rock，Klon 提供 Transparent Push |
| **Jazz** | ⭐⭐⭐⭐ | Clean Boost 模式適合 Jazz 音色 |
| **Neo Soul** | ⭐⭐⭐⭐⭐ | 溫暖、開放的音色特性 |
| **Country** | ⭐⭐⭐⭐ | Klon 側提供經典 Country Twang |

**代表性藝術家:**
- Eric Clapton（Bluesbreaker 經典用戶）
- John Mayer（Klon 經典用戶）
- Gary Clark Jr.（兩種音色都使用）
- Joe Bonamassa（Blues Rock 音色）
- Josh Scott（JHS Pedals 創辦人，推薦 BB + Klon 組合）

- 建議使用獨立供電或高品質 Isolated Power Supply

### 雙通道設計

#### **Bluesbreaker Side（BB 側）**

**電路設計改良:**
- **移除 Clipping Diodes** - 解決傳統 BB 音量低與音色暗的問題
- **提升音量與清晰度** - 避免傳統 BB 的 Dark Tone
- **Clean Control** - 源自 Cornerstone Gladio，可混入原始乾訊號

**控制旋鈕:**
1. **VOLUME** - BB 通道輸出音量
2. **DRIVE** - BB 通道增益控制
3. **TONE** - BB 通道音色調整
4. **CLEAN** - 混入乾訊號比例（Cornerstone Gladio 技術）

**音色特性:**
- 開放、清晰的 Overdrive
- 飽滿的低頻，不過度肥厚
- 平衡的中頻，不像 Tube Screamer 突出
- 清晰的高頻，避免傳統 BB 的 Dark Tone

---

#### **Klon Side（Klon 側）**

**電路設計:**
- **經典 Klon 架構** - 維持 Klon Centaur 經典音色
- **Clip Blender** - 創新功能，混合 Germanium 與 Silicon Clipping

**控制旋鈕:**
1. **VOLUME** - Klon 通道輸出音量
2. **DRIVE** - Klon 通道增益控制
3. **TONE** - Klon 通道音色調整
4. **CLIP BLENDER** - 混合 Germanium 與 Silicon Clipping

**Clip Blender 功能:**

| 設定 | Clipping 類型 | 音色特性 |
|------|--------------|---------|
| **逆時針最小** | Pure Germanium（1N34A） | 溫暖、有機，經典 Klon 音色 |
| **中間位置** | Germanium + Silicon 混合 | 平衡的音色特性 |
| **順時針最大** | Pure Silicon | 更清晰、更多 Headroom，現代音色 |

---

### Order Toggle（順序切換開關）

**功能:** 切換兩個通道的訊號順序

| 模式 | 訊號順序 | 音色特性 |
|------|---------|---------|
| **Mode 1** | BB → Klon | BB 基礎音色 + Klon 推動，溫暖飽滿 |
| **Mode 2** | Klon → BB | Klon 基礎音色 + BB 開放感，清晰透明 |

**應用建議:**
- **BB → Klon** - 適合需要更多 Gain 與飽滿度的 Lead 音色
- **Klon → BB** - 適合需要清晰度與透明感的 Rhythm 音色

---

### 4 個後方接孔（Four Rear Jacks）

**功能:** 將兩側獨立連接到 Loop Switcher 或插入其他效果器

**應用範例:**
```
訊號流程 1（獨立使用）:
Guitar → Looper Input 1 → BB Side → Looper Return 1
       → Looper Input 2 → Klon Side → Looper Return 2 → Amp

訊號流程 2（插入其他效果器）:
Guitar → BB Side Output → EQ Pedal → Klon Side Input → Amp
```

## 音色特性

### BB Side 音色特性

1. **開放、清晰的 Overdrive**
   - 移除 Clipping Diodes 後，音色更開放
   - 避免傳統 BB 的 Dark Tone 與 Low Volume 問題

2. **Clean Control 增加透明度**
   - 可混入原始乾訊號
   - 增加音色塑造彈性
   - 適合需要保留原始音色特性的演奏

3. **頻率響應**
   - **低頻:** 飽滿但不過度
   - **中頻:** 平衡，不像 Tube Screamer 突出
   - **高頻:** 開放、清晰

4. **動態反應**
   - 高 Pick Sensitivity
   - 高 Volume Sensitivity
   - 適中的 Compression

---

### Klon Side 音色特性

1. **經典 Klon 透明 Overdrive**
   - 保留吉他與音箱原音
   - 輕微提升中頻
   - 豐富的諧波泛音

2. **Clip Blender 擴展音色範圍**
   - **Germanium Mode** - 溫暖、有機，經典 Klon 音色
   - **Silicon Mode** - 清晰、現代，更多 Headroom
   - **Mixed Mode** - 連續調整兩種音色比例

3. **頻率響應**
   - **低頻:** 飽滿
   - **中頻:** 經典 Klon 中頻提升
   - **高頻:** 清晰，帶有 Sparkle

4. **動態反應**
   - 高 Pick Sensitivity
   - 高 Volume Sensitivity
   - 適中的 Compression

---

### 雙通道疊加特性

**BB → Klon 順序:**
- BB 提供基礎音色與 Body
- Klon 再次推動，增加飽滿度與 Sustain
- 適合 Lead Solo，音色溫暖飽滿

**Klon → BB 順序:**
- Klon 提供基礎透明度與清晰度
- BB 增加開放感與動態
- 適合 Rhythm，音色清晰透明

## 使用範例

### 範例 1：Blues Clean Boost（Klon Side Only）

**音樂風格:** Blues, Jazz
**適合吉他:** 任何吉他

**設定:**

| 通道 | 旋鈕 | 設定 |
|------|------|------|
| **Klon** | VOLUME | 3 點鐘（提升音量） |
| **Klon** | DRIVE | 8 點鐘（極低 Gain） |
| **Klon** | TONE | 12 點鐘 |
| **Klon** | CLIP BLENDER | 9 點鐘（偏 Germanium） |
| **BB** | 關閉 | - |

**效果:**
- 經典 Klon Clean Boost
- 輕微提升中頻與高頻
- 推動音箱進入輕度破音
- 保留吉他與音箱原音特性

**訊號鏈位置:** Guitar → Compressor → **Colosseum (Klon Only)** → Delay → Reverb → Amp

---

### 範例 2：British Rock Rhythm（BB Side Only）

**音樂風格:** Rock, Blues Rock
**適合吉他:** Humbucker 或 P90 吉他

| **BB** | TONE | 11 點鐘 |

**效果:**
- 經典 Marshall Bluesbreaker 音色
- 開放、清晰的 Overdrive
- Clean Control 保留 Clarity
- 適合 Power Chord 與 Riff

**訊號鏈位置:** Guitar → EQ → **Colosseum (BB Only)** → Delay → Reverb → Amp

## 優缺點分析

### 優點

1. **社群驅動設計確保實用性**
   - 由實際用戶投票決定功能
   - 針對傳統 BB 與 Klon 的弱點進行改良

2. **一個踏板包含兩個經典音色**
   - Marshall Bluesbreaker + Klon Centaur
   - 節省 Pedalboard 空間
   - 價格合理（$380 vs 兩個獨立 Boutique Pedal）

3. **BB 側改良解決傳統問題**
   - 移除 Clipping Diodes，提升音量與清晰度
   - Clean Control 增加透明度

4. **Klon 側 Clip Blender 增加彈性**
   - 從 Germanium 到 Silicon 連續調整
   - 涵蓋經典到現代音色

5. **Order Toggle 提供兩種不同音色層次**
   - BB → Klon（溫暖飽滿）
   - Klon → BB（清晰透明）

6. **4 個後方接孔提供極大整合性**
   - 可獨立接入 Loop Switcher
   - 可在兩側之間插入其他效果器

7. **義大利製造品質優異**
   - Cornerstone 品質保證
   - 做工精細

---

### 缺點

1. **尺寸較大**
   - 119mm × 94mm
   - Pedalboard 空間佔用較多

2. **電流消耗較高**
   - 150mA
   - 需要高品質 Power Supply

3. **學習曲線較陡**
   - 8 個旋鈕 + 1 個開關
   - 需要時間熟悉各種組合

4. **價格較高**
   - $380 USD
   - 雖然包含兩個經典音色，但對預算有限的玩家仍是負擔

5. **無獨立 LED 指示燈**
   - 無法一眼看出哪個通道開啟（需透過 Order Toggle 位置判斷）

---

### 適合人群

**最適合:**
- 需要 Bluesbreaker 與 Klon 兩種音色的玩家
- 希望節省 Pedalboard 空間的玩家
- Blues, Rock, Neo Soul 玩家
- 願意花時間深入研究的玩家
- 需要靈活 Overdrive Stacking 的玩家

**較不適合:**
- 預算有限的初學者
- Pedalboard 空間充足，偏好單功能踏板的玩家
- 需要極端失真的 Metal 玩家
- 希望即插即用、不想調整的玩家

## 總結

### 核心優勢

Cornerstone Colosseum 是市面上少數將 **Marshall Bluesbreaker** 與 **Klon Centaur** 兩大經典電路完美結合的雙通道 Overdrive 效果器。透過社群驅動的設計方式，Colosseum 不僅保留了兩種經典音色，還針對傳統弱點進行了重大改良：

- **BB 側** - 移除 Clipping Diodes，提升音量與清晰度，加入 Clean Control
- **Klon 側** - 加入 Clip Blender，從 Germanium 到 Silicon 連續調整
- **Order Toggle** - 可切換兩通道順序，創造不同音色層次
- **4 個後方接孔** - 提供極大整合性

### 最佳使用建議

1. **初學者:** 從單通道開始（BB Only 或 Klon Only），熟悉各自音色特性
2. **中階使用者:** 實驗雙通道疊加，嘗試不同 Order Toggle 設定
3. **進階使用者:** 利用 4 個後方接孔整合到複雜的訊號鏈中，建立完整的 Overdrive Stacking 系統

### 訊號鏈定位

**最佳位置:** Signal Chain 中段（Pre-Amp Gain Section）

**推薦配置:**
```
Guitar → Compressor → EQ → Colosseum → (其他 Overdrive) → Modulation → Delay → Reverb → Amp
```

**Overdrive Stacking 範例:**
```
範例 1（Clean Boost + Stacking）:
Soul Food (Clean Boost) → Colosseum (BB→Klon) → Amp

範例 2（三級疊加）:
Morning Glory (Low Gain) → Colosseum Klon Side (Mid Gain) → OCD (High Gain) → Amp

範例 3（獨立使用）:
Guitar → Looper Send 1 → Colosseum BB Side → Looper Return 1
       → Looper Send 2 → Colosseum Klon Side → Looper Return 2 → Amp
```
