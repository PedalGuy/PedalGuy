## 器材概述

### 基本資訊

**品牌:** Cornerstone Music Gear
**型號:** Nucleo
**全名:** Cornerstone Nucleo Stereo Reverb (Paul Davids Signature)
**類型:** 數位立體聲 Reverb
**設計國:** 義大利
**製造:** 義大利手工製作
**設計合作夥伴:** Paul Davids (荷蘭吉他手/YouTuber)
**發表年份:** 2024 年
**開發時間:** 3 年
**音色引擎:** M7 Cortex Engine
**靈感來源:** 奧地利 Zwentendorf 核電廠自然殘響
**價格:** $350 USD / €359 EUR

### 核心賣點

Cornerstone Nucleo 是荷蘭吉他手兼 YouTuber **Paul Davids** 與義大利手工效果器品牌 **Cornerstone Music Gear** 合作開發的夢想 Reverb 效果器。這款效果器的誕生源於一次特殊的經歷：Paul Davids 走進奧地利 **Zwentendorf 核電廠** (一座耗資十億歐元建造但從未啟用的 1970 年代核電廠)，在這個巨大、共鳴的空間中彈奏吉他，創造出令人著迷的音色體驗。

Nucleo 透過 **Impulse Response** 技術捕捉這個獨特空間的殘響特性，並結合 **M7 Cortex Engine** 的強大運算能力，提供從親密控制室到無限大廳的極端音色範圍。經過 **3 年開發**，Nucleo 成為 Paul Davids 口中「取代 BigSky 的夢想 Reverb」。

**獨特之處:**
- **核電廠反應爐空間感** - 真實錄製 Zwentendorf 核電廠 Impulse Response
- **三種 Reverb 引擎** - Room / Hall / Reactor (靈感來自核電廠不同區域)
- **三種 Voicing 選項** - Normal / Vintage / Lo-Fi
- **最長 90 秒 Decay** - 史詩般的延續時間
- **128 個 MIDI 預設** - 3 個機身預設 + 125 個 MIDI 可調用
- **Freeze 功能** - 凍結殘響創造無限 Pad，支援 Latch Mode 與 Freeze-Pitch 調整
- **立體聲輸出** - Right Output 採用 Transformer-Isolated 避免 Ground Loop
- **USB Audio Interface** - 可作為電腦音訊介面直接錄製

### Reverb 引擎架構

Nucleo 搭載 **M7 Cortex Engine**，這是高階數位 Reverb 常用的運算引擎 (如 Strymon BigSky/NightSky)。M7 提供極高的運算能力，能夠創造從細膩到極端的各種殘響效果。

#### **三種 Reverb 模式**

##### **1. Room (控制室)**

**靈感來源:** 核電廠控制室
**空間特性:** 親密、反射清晰的小空間

**音色特性:**
- 早期反射音明顯
- 殘響時間短至中等
- 適合需要清晰度與定位感的音樂

**適合:**
- Jazz 細膩演奏
- Neo Soul 和弦
- Blues Lead
- 需要保持音符清晰度的場合

---

##### **2. Hall (大廳/冷卻室)**

**靈感來源:** 核電廠冷卻室
**空間特性:** 寬廣、開闊的大廳空間

**音色特性:**
- 寬廣的空間感
- 殘響時間中至長
- 平衡的頻率響應

**適合:**
- Post Rock Lead
- Worship 和弦
- Classic Rock Solo
- 需要寬廣空間感但不過度誇張的場合

---

##### **3. Reactor (反應爐核心)**

**靈感來源:** 核電廠反應爐核心
**空間特性:** 神秘、大氣、巨大、膨脹的空間

**音色特性:**
- 極端寬廣的空間感
- 電影般的質感
- 巨大、膨脹的聲音
- 創造「你感覺它在你周圍，而不是從音箱出來」的體驗

**適合:**
- Ambient 音景
- Post Rock 高潮段落 (Climax)
- Shoegaze 音牆
- 電影配樂般的吉他音色

**Paul Davids 原話:**
> "Usually you hear it coming out of your amp. Here, you feel it all around you."
> (通常你聽到聲音從音箱出來，但在這裡，你感覺它在你周圍。)

---

#### **三種 Voicing 選項**

每個 Reverb 模式都可搭配三種 Voicing，創造 3 × 3 = 9 種基礎音色組合。

##### **1. Normal (正常)**

**特性:**
- 清晰、現代、精準
- 高保真度
- 頻率響應平衡

**適合:**
- 需要清晰度的音樂
- 現代 Pop/Rock
- Studio Recording

---

##### **2. Vintage (復古)**

**特性:**
- 溫暖、有機
- 略帶 Wow and Flutter (磁帶不穩定感)
- 高頻略微圓滑

**適合:**
- Neo Soul
- Blues
- Classic Rock
- 追求類比感的玩家

---

##### **3. Lo-Fi (低保真)**

**特性:**
- 復古、粗糙
- 明顯的 Bit-Crushing 或 Sample Rate Reduction 效果
- 顆粒感、質感豐富

**適合:**
- Shoegaze
- Experimental
- Indie Rock
- 追求 Lo-Fi 美學的玩家

### 控制旋鈕詳解

Nucleo 的控制介面採用 **雙功能旋鈕設計**，每個旋鈕都有兩個功能，透過模式切換使用。

#### **主要旋鈕 (第一功能)**

1. **BLEND (乾濕混合)**
   - **功能:** 原始訊號與 Reverb 訊號混合比例
   - **範圍:** 0% ~ 100%

2. **MOD (調變深度)**
   - **功能:** 控制殘響音的調變程度
   - **效果:** 從細微調變到完整調變景觀

3. **AIR (Shimmer 高八度)**
   - **功能:** Shimmer 高八度效果 (或低八度)
   - **效果:** 添加高八度閃爍感，創造天使般音色

4. **FLUX (顆粒質感)**
   - **功能:** 顆粒質感、Geiger Counter 風格隨機聲響
   - **效果:** 創造實驗性、隨機脈衝音色

5. **DECAY (殘響長度)**
   - **功能:** 控制 Reverb 尾音持續時間
   - **範圍:** 短 ~ 90 秒 (epic decay)

6. **TONE (音色)**
   - **功能:** 控制殘響音色明暗
   - **範圍:** 溫暖 ~ 明亮

---

#### **第二功能 (按住特定按鈕切換)**

1. **VOLUME (整體音量)**
   - **對應:** BLEND 旋鈕第二功能

2. **RATE (調變速率)**
   - **對應:** MOD 旋鈕第二功能
   - **功能:** 控制調變速度

3. **PITCH (Shimmer 音高)**
   - **對應:** AIR 旋鈕第二功能
   - **功能:** 控制 Shimmer 音高 (高八度或低八度)

4. **SPEED (顆粒速度)**
   - **對應:** FLUX 旋鈕第二功能
   - **功能:** 控制顆粒效果速度

5. **PRE-D (Pre-Delay)**
   - **對應:** DECAY 旋鈕第二功能
   - **功能:** 控制殘響出現前的延遲時間

6. **DIFF (Diffusion)**
   - **對應:** TONE 旋鈕第二功能
   - **功能:** 控制殘響密度與擴散程度

---

### Freeze 功能詳解

Nucleo 的 Freeze 功能極為強大，提供多種模式與控制選項。

#### **基本 Freeze 功能**

**功能:**
- 凍結當前殘響尾音並無限延續
- 可在凍結音上繼續演奏

**使用方式:**
1. 演奏和弦或單音
2. 按下 Freeze Footswitch
3. 該音符/和弦的殘響凍結並無限延續
4. 在 Pad 上繼續演奏

---

#### **Latch Mode (閂鎖模式)**

**功能:**
- Freeze 狀態持續保持，直到再次按下 Footswitch
- 適合需要長時間 Pad 的場合

**應用:**
- Ambient 無限音景
- Worship 演奏時建構持續 Pad
- Post Rock 段落轉換

---

#### **Freeze-Pitch (凍結音高調整)**

**功能:**
- 調整凍結音的音高
- 創造和聲變化或實驗性音色

**控制旋鈕:** FREEZE-PITCH (獨立旋鈕)

**應用:**
- 將凍結的 C Major 和弦音高上升 +5 半音 → F Major
- 創造和聲進行
- 實驗性音色設計

---

#### **Freeze-Volume (凍結音量)**

**功能:**
- 控制凍結音的音量
- 平滑淡入淡出凍結 Pad

**控制旋鈕:** FREEZE-VOLUME (獨立旋鈕)

### 連接規格

| 連接埠 | 規格 | 說明 |
|--------|------|------|
| **INPUT** | 6.3mm TRS Stereo Jack | 支援 Mono 或 Stereo 輸入 |
| **OUTPUT L** | 6.3mm Jack | 立體聲左聲道輸出 |
| **OUTPUT R** | 6.3mm Jack | 立體聲右聲道輸出 (**Transformer-Isolated**) |
| **MIDI IN** | 3.5mm TRS | MIDI 輸入 (非標準 DIN 5-pin) |
| **MIDI OUT** | 3.5mm TRS | MIDI 輸出 (非標準 DIN 5-pin) |
| **USB-C** | USB-C Port | Audio Interface 功能 + 韌體更新 |
| **DC 9V** | 2.1mm Jack | Center Negative |

**重要:**
- **Right Output 採用 Transformer-Isolated 技術** - 有效避免 Ground Loop 雜訊
- **USB-C 支援 Audio Interface 功能** - 可直接錄製 Reverb 效果至 DAW

---

### 特殊技術

#### **Transformer-Isolated Right Output**

**問題:** 立體聲配置中，兩個輸出連接不同音箱時容易產生 Ground Loop 雜訊
**解決方案:** Right Output 採用 Transformer-Isolated 技術，隔離接地迴路
**效果:** 消除 Hum 雜訊，提供乾淨的立體聲輸出

---

#### **USB Audio Interface 功能**

**功能:**
- Nucleo 可作為電腦音訊介面
- 直接錄製 Reverb 效果至 DAW
- 韌體更新

**應用:**
- Studio Recording 直接錄製
- 免除額外 Audio Interface
- 保持效果器音色完整性

## 音色特性

### 核心音色特徵

1. **寬廣、豐富、大氣**
   - 從親密到無限的空間感範圍
   - 電影般質感

2. **有機、會呼吸**
   - M7 Cortex Engine 創造自然殘響
   - MOD 功能添加微妙調變

3. **Reactor 模式的獨特性**
   - 神秘、大氣的殘響
   - 巨大、膨脹的聲音
   - 創造「你感覺它在你周圍」的體驗
   - 靈感來自真實核電廠空間

4. **三種 Voicing 提供多樣性**
   - **Normal** - 清晰、現代
   - **Vintage** - 溫暖、Wow and Flutter
   - **Lo-Fi** - 復古、粗糙、顆粒感

---

### 頻率響應特性

**可透過 TONE/DIFF 控制調整:**

| 模式 | 低頻 | 中頻 | 高頻 | 適合 |
|------|------|------|------|------|
| **Room + Normal** | 平衡 | 清晰 | 明亮 | Jazz, Neo Soul |
| **Hall + Vintage** | 溫暖 | 豐富 | 圓滑 | Blues, Classic Rock |
| **Reactor + Lo-Fi** | 厚實 | 粗糙 | 顆粒感 | Shoegaze, Ambient |

---

### 調變特性

**MOD (Modulation) 控制:**
- **低設定 (10-20%):** 細微調變，增加有機感
- **中設定 (30-50%):** 明顯調變，創造運動感
- **高設定 (60%+):** 完整調變景觀，實驗性音色

**RATE (Modulation Rate):**
- 控制調變速度
- 慢速 → Vintage Tape 感
- 快速 → Chorus/Vibrato 感

---

### AIR (Shimmer) 特性

**功能:**
- 添加高八度 (或低八度) Shimmer 效果
- PITCH 控制可調整音高

**設定建議:**
- **Hall/Reactor + AIR (高八度)** → 天使般 Shimmer
- **Room + AIR (低八度)** → 深沉和聲支撐
- **搭配 Freeze** → 凍結 Shimmer Pad

---

### FLUX (顆粒質感) 特性

**功能:**
- 顆粒質感、Geiger Counter 風格隨機聲響
- 創造實驗性音色

**設定建議:**
- **低設定:** 細微的隨機脈衝
- **高設定:** 明顯的顆粒破碎效果
- **搭配 Reactor Mode** → 詭異音景

## 使用範例

### 範例 1：Room Mode - Jazz Clean Tone

**音樂風格:** Jazz, Neo Soul
**適合吉他:** Semi-Hollow、Single-Coil 吉他

**設定:**

| 參數 | 設定 |
|------|------|
| **Mode** | Room (控制室) |
| **Voicing** | Normal (清晰) |
| **BLEND** | 30-40% (細膩) |
| **DECAY** | 短至中 (1-2 秒) |
| **TONE** | 11 點鐘 (略明亮) |
| **PRE-D** | 短 (20-40ms) |
| **MOD** | 極低 (5-10%) |
| **AIR** | 關閉 |
| **FLUX** | 關閉 |

**效果:**
- 親密、清晰的空間感
- 不掩蓋音符細節
- 適合和弦演奏與 Arpeggios

**訊號鏈位置:**
```
Guitar → Compressor → EQ → Nucleo (Room) → Amp
```

### 範例 3：Freeze Pad - Worship 和弦演奏

**音樂風格:** Worship, Ambient
**適合吉他:** 任何吉他

**設定:**

| 參數 | 設定 |
|------|------|
| **Mode** | Hall (大廳) |
| **Voicing** | Normal (清晰) |
| **BLEND** | 70% (高 Reverb) |
| **DECAY** | 長 |
| **TONE** | 12 點鐘 |
| **AIR** | 中 (Shimmer) |
| **Freeze-Volume** | 中至高 |
| **Freeze-Pitch** | 0 (不變) |

**技巧:**
1. 演奏開放和弦 (如 C Major)
2. **按下 Freeze Footswitch**
3. 和弦的 Shimmer Reverb 凍結並無限延續
4. 在 Pad 上繼續演奏
5. 切換至新和弦前，再次按 Freeze 清除舊 Pad

**效果:**
- 持續的和聲 Pad 支撐
- Shimmer 添加豐富層次
- Latch Mode 讓 Pad 持續直到手動關閉

**應用場景:**
- Worship 演奏
- Ambient 和弦進行
- Post Rock 段落轉換

### 範例 5：Freeze-Pitch Harmony Pad

**音樂風格:** Ambient, Experimental
**適合吉他:** 任何吉他

**設定:**

| 參數 | 設定 |
|------|------|
| **Mode** | Hall (大廳) |
| **Voicing** | Normal |
| **BLEND** | 80-90% |
| **DECAY** | 長 |
| **AIR** | 高 (Shimmer) |
| **Freeze-Pitch** | **+5 半音** |
| **Freeze-Volume** | 高 |

**技巧:**
1. 演奏 C Major 和弦
2. **按下 Freeze Footswitch**
3. 和弦凍結
4. **調整 Freeze-Pitch 至 +5 半音**
5. 凍結音變成 F Major 和聲
6. 繼續演奏 C Major，形成 C → F 和聲進行

**效果:**
- Freeze-Pitch 創造和聲變化
- 不需實際演奏即可建構和聲進行
- 極具創意的 Pad 建構方式

**應用場景:**
- Ambient 和聲實驗
- 現場演出即興和聲
- Studio 錄音創意層次

## 總結

### 核心優勢

Cornerstone Nucleo 是 Paul Davids 與 Cornerstone Music Gear 合作開發的夢想 Reverb 效果器，透過真實核電廠 Impulse Response 創造獨特的空間感。**三種 Reverb 模式**、**三種 Voicing 選項**、**強大的 Freeze 功能**、**128 個 MIDI 預設** 以及 **USB Audio Interface 功能** 讓這款效果器成為 High-End Reverb 市場的強力競爭者。

**Reactor 模式** 創造的巨大、膨脹空間感極具電影感，適合 Post Rock 與 Ambient 玩家。**Right Output Transformer-Isolated** 技術有效避免 Ground Loop 雜訊，提供乾淨的立體聲輸出。

### 最佳使用建議

1. **初學者:** 從 Room Mode + Normal Voicing 開始，BLEND 保持 30-40%
2. **中階使用者:** 實驗 Reactor Mode 與 Freeze 功能，建立 Ambient Pad
3. **進階使用者:** 使用 Freeze-Pitch 創造和聲進行，利用 USB Audio Interface 直接錄製，建立完整的 128 預設庫

### 訊號鏈定位

**最佳位置:** FX Loop 或 Post-Delay (訊號鏈最後階段)

**推薦配置 (Stereo):**
```
Guitar → Compressor → Overdrive → Delay → Nucleo (Stereo) → Stereo Amp (L/R)
```

**Ambient 配置 (with Freeze):**
```
Guitar → Clean Boost → Delay → Nucleo (Reactor, Freeze) → Stereo Amp
```

**Post Rock 配置:**
```
Guitar → Overdrive → Delay → Nucleo (Reactor, Long Decay) → Stereo Amp
```
