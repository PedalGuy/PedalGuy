## 器材概述

### 基本資訊

**品牌:** Lichtlaerm Audio
**型號:** As Above So Below
**全名:** As Above So Below - Cinematic Octave Divider Reverb
**類型:** Shimmer Reverb + Octave Divider
**設計國:** 德國 (柏林)
**發表年份:** 2024 年 (原型開發始於 2018 年)
**價格:** $225 USD

### 核心賣點

Lichtlaerm Audio As Above So Below 是一款突破性的「雙向八度 Shimmer Reverb」效果器，獨特之處在於提供**高八度 (Above)**、**低八度 (Below)** 以及**雙向八度 (Both)** 三種模式，創造從天使般飄浮到地底深沉 Drone 的極端音色。

這款效果器特別針對**高增益音色**優化，能在 Overdrive/Distortion 之後產生清晰、可辨的 Shimmer 效果，而不會泥濘混濁。設計理念「As Above, So Below」(上如其下) 象徵天堂與地底的對立統一，透過音色創造哲學性的對話。

**獨特之處:**
- **雙向八度架構** - Above (高八度) / Below (低八度) / Both (雙向)
- **高增益友善** - 專門設計可搭配失真效果使用
- **Freeze 功能** - 凍結殘響創造無限 Ambient Pad
- **FX Loop** - 將殘響路由至外部效果器進行創意處理
- **Expression Pedal 支援** - 即時控制八度音量淡入淡出
- **最高 +6dB Boost** - LEVEL 控制可提升整體音量

### 八度分頻架構

#### **三種模式詳解**

##### **1. Above Mode (高八度模式)**

**音色特性:**
- 天使般、飄浮如光環
- 高八度音與殘響結合，創造閃爍 (Shimmer) 效果
- 適合 Post Rock、Ambient Lead

**控制參數:**
- **X Control (Pre-Delay):** 控制殘響與高八度出現的延遲時間
- **Y Control (Octave Volume):** 控制高八度音量
- **DAMPEN:** 控制高頻衰減 (玻璃般 or 復古溫暖)

**應用場景:**
- Clean Tone Arpeggios
- Post Rock Swells
- Ambient Lead Guitar
- Worship Pad

---

##### **2. Below Mode (低八度模式)**

**音色特性:**
- 深沉、Drone、Sub-Octave 質感
- 低八度音可透過 X Control 時間偏移
- 創造洞穴般、膨脹的空間感
- 適合 Doom、Drone、Ambient Bass

**控制參數:**
- **X Control (Pre-Delay + Time Offset):** 控制低八度出現時間與偏移量
- **Y Control (Octave Volume):** 控制低八度音量
- **DECAY:** 長 Decay 創造無盡 Drone

**應用場景:**
- Doom Metal Sub-Octave Riff
- Drone 底層建構
- Ambient 深沉音景
- Post Rock 低頻支撐

---

##### **3. Both Mode (雙向八度模式)**

**音色特性:**
- **天堂與地底碰撞** - 高八度與低八度同時存在
- 詭異和聲豐富度
- 洗刷質感 (Wash Texture)
- 極端實驗性音色

**控制參數:**
- **X Control (低八度音量):** 獨立控制低八度音量
- **Y Control (高八度音量):** 獨立控制高八度音量
- **混合比例可調整:** 創造從天堂主導到地底主導的各種平衡

**應用場景:**
- 實驗性音樂
- Ambient 極端音景
- Post Rock 高潮段落 (Climax)
- Drone 豐富和聲層次

### 控制旋鈕詳解

#### **主要旋鈕**

1. **LEVEL (輸出電平)**
   - **功能:** 總輸出音量控制
   - **範圍:** 0dB ~ +6dB Boost
   - **應用:** 可用作 Clean Boost 同時添加 Shimmer 效果

2. **MIX (乾濕混合)**
   - **功能:** 原始訊號與 Reverb 訊號混合比例
   - **範圍:** 0% ~ 100%
   - **建議:**
     - **20-40%** - 細膩 Shimmer 層次
     - **50-70%** - 明顯 Shimmer 效果
     - **80-100%** - Ambient Pad 感覺

3. **DECAY (殘響長度)**
   - **功能:** 控制 Reverb 尾音持續時間
   - **範圍:** 短 ~ 長 (無限延續需搭配 Freeze)
   - **應用:** 長 Decay + Below Mode = 無盡 Drone

4. **DAMPEN (高頻衰減)**
   - **功能:** 控制殘響音色明亮度
   - **調整影響:**
     - **低設定 (保留高頻)** → Glassy Shimmer (玻璃般清晰)
     - **高設定 (削減高頻)** → Vintage Bloom (復古溫暖)
   - **應用:** 搭配高增益時略提高 DAMPEN 避免刺耳

5. **X Control (動態功能)**
   - **Above/Below Mode:** Pre-Delay (殘響出現前的延遲時間)
   - **Both Mode:** 低八度音量控制

6. **Y Control (動態功能)**
   - **Above/Below Mode:** 八度音量控制
   - **Both Mode:** 高八度音量控制

---

### 特殊功能

#### **1. Freeze Function (凍結功能)**

**功能:**
- 按住 Freeze Footswitch，當前殘響無限延續
- 創造 Glacial、Eternal 質感的 Ambient Pad
- 可在凍結音上繼續演奏

**使用方式:**
1. 演奏和弦或單音
2. 按住 Freeze Footswitch
3. 該音符/和弦的殘響凍結並無限延續
4. 在 Pad 上繼續演奏

**應用場景:**
- Post Rock 段落轉換 (Transition)
- Ambient 無限音景
- Worship 演奏時建構 Pad
- Solo 時的和聲支撐

---

#### **2. FX Loop (效果器迴路)**

**功能:**
- 將殘響訊號路由至外部效果器
- Send/Return 配置

**創意應用:**
1. **殘響 → Distortion** - 扭曲殘響尾音，創造實驗性音色
2. **殘響 → Modulation (Phaser/Flanger)** - 調變殘響創造獨特空間感
3. **殘響 → Delay** - 延遲殘響創造複雜層次
4. **殘響 → Tremolo** - 脈動式殘響效果

**範例配置:**
```
Guitar → As Above So Below [SEND] → Distortion → [RETURN] → Amp
```
→ 殘響音經過失真處理，創造 Shoegaze 風格音牆

---

#### **3. Expression Pedal Input (表情踏板輸入)**

**功能:**
- 即時控制 Y Control (八度音量)
- 透過腳踏板動態控制高/低八度淡入淡出

**應用場景:**
- **Above Mode:** 即時淡入高八度 Shimmer (從 Clean 到 Shimmer Lead)
- **Below Mode:** 即時淡入低八度 Drone (從正常到深沉)
- **Both Mode:** 控制高八度音量，動態改變 Shimmer 強度

**範例:**
- Solo 起始段落 Expression Pedal 在 0%，逐漸推到 100% 創造漸強 Shimmer 效果

---

#### **4. Ramp Footswitch (漸變踏板)**

**功能:**
- 支援 Ramp 功能外部踏板輸入
- 可能用於自動化參數變化 (待手冊確認)

---

### Bypass 技術

**規格:**
- **Smart Relay Bypass Technology**
- **Buffered Bypass** (Always-on Buffer)
- **Spillover 可切換** - 切換效果器時殘響音自然消失或立即中斷

**優點:**
- Spillover 功能讓現場演出切換更自然
- Buffered Bypass 保護訊號完整性

## 音色特性

### 核心音色特徵

1. **寬廣、膨脹的空間感**
   - 洞穴般、電影般的殘響質感
   - 適合創造 Cinematic Soundscapes

2. **Above (高八度) - 天使般飄浮**
   - 閃爍 (Shimmer) 效果明顯
   - 如光環般飄浮在音符之上
   - 適合 Post Rock 與 Worship

3. **Below (低八度) - 深沉 Drone**
   - Sub-Octave 質感
   - 低八度可時間偏移，創造錯位感
   - 適合 Doom Metal 與 Drone 音樂

4. **Both (雙向八度) - 天堂與地底碰撞**
   - 詭異和聲豐富度
   - 洗刷質感 (Wash Texture)
   - 極端實驗性音色

---

### DAMPEN 控制的音色變化

**DAMPEN 旋鈕是塑造 Shimmer 質感的關鍵:**

| 設定 | 高頻處理 | 音色特性 | 適合 |
|------|---------|---------|------|
| **低 (保留高頻)** | 高頻完整保留 | Glassy Shimmer (玻璃般) | Clean Tone, Ambient |
| **中 (適度衰減)** | 高頻略微圓滑 | 平衡 Shimmer | 通用設定 |
| **高 (大幅衰減)** | 高頻明顯削減 | Vintage Bloom (復古) | 高增益, Shoegaze |

---

### 高增益友善特性

**As Above So Below 特別設計可搭配高增益使用:**

**問題:** 大多數 Shimmer Reverb 在高增益下會變得泥濘、混濁
**解決方案:** Lichtlaerm 調整演算法，確保 Shimmer 在 Distortion 之後仍清晰可辨

**建議設定 (高增益):**
- **Mode:** Above 或 Both
- **MIX:** 40-60% (不要過高)
- **DECAY:** 中等 (避免過長尾音累積)
- **DAMPEN:** 略高 (控制高頻刺耳)

**訊號鏈:**
```
Guitar → Overdrive/Distortion → As Above So Below → Amp
```

## 使用範例

### 範例 1：Post Rock Shimmer Lead (Above Mode)

**音樂風格:** Post Rock, Ambient
**適合吉他:** 任何吉他

**設定:**

| 參數 | 設定 |
|------|------|
| **Mode** | Above (高八度) |
| **LEVEL** | 12 點鐘 (Unity Gain) |
| **MIX** | 50-60% (明顯 Shimmer) |
| **DECAY** | 2 點鐘 (長殘響) |
| **DAMPEN** | 10 點鐘 (保留高頻) |
| **X (Pre-Delay)** | 9 點鐘 (短 Pre-Delay) |
| **Y (Octave Volume)** | 1 點鐘 (明顯高八度) |

**效果:**
- 高八度音與殘響結合，創造天使般飄浮感
- 適合 Lead Solo 或 Arpeggios
- Glassy Shimmer 質感，清晰不混濁

**訊號鏈位置:**
```
Guitar → Compressor → Overdrive (輕度) → As Above So Below → Delay → Amp
```

**應用場景:**
- Post Rock Solo
- Ambient Lead Guitar
- Clean Tone Arpeggios

## 優缺點分析

### 優點

1. **雙向八度架構獨特**
   - 市面上少數提供 Above/Below/Both 三種模式的效果器
   - 創造極端音色範圍 (天堂到地底)

2. **高增益友善設計**
   - 特別優化可搭配 Distortion/Fuzz 使用
   - Shimmer 在高增益下仍清晰不混濁

3. **Freeze 功能實用**
   - 創造無限 Ambient Pad
   - 適合現場演出與錄音

4. **FX Loop 創意潛力**
   - 可將殘響路由至外部效果器
   - 創造實驗性音色

5. **Expression Pedal 支援**
   - 即時控制八度音量
   - 極具表現力

6. **價格合理**
   - $225 USD，相較其他 Shimmer Reverb 平價
   - 性價比極高

7. **LEVEL 控制可 Boost**
   - 最高 +6dB Boost
   - 可當 Clean Boost 使用

---

### 缺點

1. **Mono 輸出限制**
   - 僅提供 Mono 輸出，無 Stereo 選項
   - 對追求 Stereo Shimmer 的玩家是缺點

2. **僅支援 9V DC**
   - 不支援 12V/18V 供電
   - 限制與部分 Power Supply 的兼容性

3. **X/Y 功能依模式而異**
   - 在不同模式下 X/Y Control 功能不同
   - 可能造成初學者混淆

4. **品牌知名度較低**
   - Lichtlaerm Audio 是較小眾品牌
   - 資訊與評測相對較少

5. **學習曲線**
   - 三種模式與動態控制需要時間熟悉
   - 不適合希望即插即用的玩家

---

### 適合人群

**最適合:**
- Post Rock, Ambient, Shoegaze 玩家
- 追求 Shimmer Reverb 的玩家
- 需要高增益友善 Reverb 的 Metal 玩家
- Doom/Drone 音樂創作者
- 追求實驗性音色的玩家
- 預算有限但需要高品質 Shimmer 的玩家

**較不適合:**
- 追求 Stereo Shimmer 的玩家 (考慮 Cornerstone Nucleo)

## 總結

### 核心優勢

Lichtlaerm Audio As Above So Below 是市面上少數提供**雙向八度架構**的 Shimmer Reverb，獨特的 Above/Below/Both 三種模式讓玩家可創造從天使般飄浮到地底深沉 Drone 的極端音色。特別針對**高增益音色優化**，讓 Shimmer 效果在 Distortion 之後仍清晰可辨，這是其他 Shimmer Reverb 難以達成的。

**Freeze 功能**、**FX Loop**、**Expression Pedal 支援** 以及 **+6dB Boost** 讓這款效果器在 $225 USD 的價位上提供極高的性價比。

### 最佳使用建議

1. **初學者:** 從 Above 模式開始，MIX 保持 40-50%，DAMPEN 中性設定
2. **中階使用者:** 實驗 Below 模式與 Freeze 功能，建立 Ambient Pad
3. **進階使用者:** 使用 Both 模式創造實驗性音色，利用 FX Loop 路由殘響至外部效果器，Expression Pedal 動態控制

### 訊號鏈定位

**最佳位置:** FX Loop 或 Post-Gain (Overdrive/Distortion 之後)

**推薦配置:**
```
Guitar → Compressor → Overdrive/Distortion → As Above So Below → Delay → Amp
```

**高增益配置:**
```
Guitar → Fuzz → As Above So Below (DAMPEN 略高) → Amp
```

**Ambient 配置 (with Freeze):**
```
Guitar → Clean Boost → As Above So Below (Both Mode, Freeze) → Stereo Delay → Amp
```
